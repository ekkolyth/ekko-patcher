# Ekko Patcher

A fork of [RomPatcher.js](https://github.com/marcrobledo/RomPatcher.js/), made for Cross Platform Installation (Windows, MacOS, Linux)

Supported formats:

- IPS
- UPS
- APS (N64/GBA)
- BPS
- RUP
- PPF
- EBP
- BSDiff (.bdf, .bspatch)
- Paper Mario Star Rod (.mod)
- VCDiff (.xdelta, .vcdiff)

Features:

- can patch and create patches
- shows ROM CRC32, MD5 and SHA-1 before patching
- can remove headers before patching
- unzips files automatically
